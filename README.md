## Kasih Collection
## Pra UTS Praktikum Sistem Informasi Berbasis Web

Aplikasi e-commerce sederhana berbasis Laravel yang memiliki fitur admin dan user, termasuk pengelolaan produk, kategori, serta keranjang belanja.

---

## Fitur

### Admin
* CRUD Produk
* Mengelola kategori
* Upload gambar produk

### User
* Melihat produk
* Melihat kategori
* Menambahkan ke keranjang

### Keranjang
* Tambah produk
* Hapus produk
* Lihat isi keranjang
